Importando Bootstrap

Para importar o Bootstrap, pode ser via link sem download (no momento de execução do arquivo, deve-se ter internet) ou arquivo local baixado do site (recomendado).

Usando a forma local

Vá até o link de download do Bootstrap clicando aqui

Após baixar, descompacte o arquivo e procure por bootstrap.min.css e bootstrap.min.js

O arquivo CSS deve ser vinculado à página HTML via link e o arquivo JS, via script. Isto feito, o Bootstrap está instalado!
